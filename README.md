# Assignment-3
Game Idea

A classic 3×3 Tic-Tac-Toe game where two players take turns placing X and O on the board. The goal is to form a straight line of three symbols before the opponent does.


---

Rules

Player X always starts first.

Players pick positions 1–9 based on the displayed board.

A position once taken cannot be used again.

The first player to align three marks (row, column, or diagonal) wins.

If all 9 cells are filled without a winning line, the game ends in a draw.



---

Controls

Input a number from 1 to 9 to place your symbol in the corresponding cell.

Invalid/occupied positions will prompt a retry.



---

Features

Clean 3×3 board display.

Turn-based 2-player gameplay.

Automatic win checking (rows, columns, diagonals).

Draw detection when board is full.

Input validation for safe play.
